# healthlab
This project developed with Laravel 5.5.

## This have follwing features:
* Search Doctor 
* Predict Disease
* Signin/ Signup
* Add Diasese 
* Add Symptom
* and many more

# Installation

`git clone https://github.com/shafikshaon/healthlab.git`

`php artisan key:generate`

Create data base `healthlab`

Then `php artisan migrate --seed`
